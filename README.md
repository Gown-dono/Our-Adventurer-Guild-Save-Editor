# Our Adventurer Guild Save Editor

Closed-source Windows x64 binary distribution for the Our Adventurer Guild Save Editor.

## Download And Run

1. Download the repository as a ZIP file or clone it.
2. Open `OagSaveEditor`.
3. Run `OagSaveEditor.exe`.

The app is published as a self-contained Windows build, so users do not need to install the .NET runtime separately.

## Notes

- The editor works with local save files only.
- Create or keep backups before editing saves.
- Source code and debug symbols are intentionally not included in this repository.
